To reproduce the project:
  1. Download raw data
  2. Open process_WHO.ipynb using Google Colab
  3. Upload ghe2021_deaths_whoregion_new2.xlsx to Google Colab
  4. Run process_WHO.ipynb -> gives you one part of the processed data
  5. Open process_UN.ipynb using Google Colab
  6. Upload WPP2024_MORT_F07_3_ABRIDGED_LIFE_TABLE_FEMALE.xlsx & WPP2024_MORT_F07_2_ABRIDGED_LIFE_TABLE_MALE.xlsx to Google Colab
  7. Run process_WHO.ipynb -> gives you the remaining part of the processed data
  8. There is no need to upload the processed data to Observable. They are already stored there
